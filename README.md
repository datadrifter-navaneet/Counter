A beautiful, responsive counter application built with pure HTML, CSS, and vanilla JavaScript. Features increment, decrement, and reset functionality with smooth animations and color-coded display.

Features
--------
- Increment (+), Decrement (-), and Reset buttons
- Real-time counter display with color coding (green=positive, red=negative)
- Smooth hover animations and button effects
- Fully responsive design for mobile and desktop
- Modern glassmorphism UI with gradients
- No dependencies - pure vanilla code

Demo
----
Live Demo: https://yourusername.github.io/simple-counter/ (Deploy via GitHub Pages)

Tech Stack
----------
- Frontend: HTML5, CSS3 (Grid/Flexbox), Vanilla JavaScript
- Styling: CSS Animations, Gradients, Glassmorphism
- No external libraries or frameworks

Getting Started
---------------
1. Clone/Download the repo
2. Open `index.html` in any modern browser
3. Start counting! No build step required

Usage
-----
- Click + to increment counter
- Click - to decrement counter  
- Click Reset to return to 0
- Counter color changes based on value (green+, red-, black=0)
